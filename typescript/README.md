# Gilded Rose

This is the Gilded Rose kata in TypeScript with Jest

## Getting started

Install dependencies

```sh
npm install
```

## Running tests

To lint

```sh
npm run lint
```

To run all tests

```sh
npm run test
```

To run all tests in watch mode

```sh
npm run test:watch
```

To generate test coverage report

```sh
npm run test:coverage
```
